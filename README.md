Dependencies
------------

This game requires Python 3.  I recommend installing the dependencies (Panda3D 1.10, esper) via pip, using the command:

```
pip install -r requirements.txt
```

Running the game
----------------

Open a terminal window, "cd" to the game directory and run:

```
python run_game.py
```
